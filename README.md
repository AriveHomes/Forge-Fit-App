# ForgeFit Studio 5.1 — Self-Contained GitHub Build

This version fixes the broken exercise images shown in the previous GitHub upload. Every image, style, and script is embedded directly inside `index.html`, so there is no `assets` folder to forget or upload incorrectly.

## Upload
1. In the GitHub repository, delete the old files (especially `sw.js`).
2. Upload this new `index.html` to the repository root.
3. Wait for GitHub Pages to deploy.
4. Open the site in a Private Safari tab first. You should see a small `5.1` badge beside the ForgeFit Studio logo.
5. Remove the old Home Screen shortcut and add the site again.

This is a working front-end prototype. Production App Store release still requires secure accounts/cloud sync, original or licensed exercise media, privacy disclosures, billing, and backend services.
